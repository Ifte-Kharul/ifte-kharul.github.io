## What it does

The Researcher lets users attach notes directly to webpages and see them again whenever the same page is opened with the same browser profile.

## Why it is useful

For research, study, and repeated reading workflows, context-specific notes reduce switching cost between documents and external apps.

## Key capabilities

- Add sticky notes on any webpage.
- Store notes locally in the browser profile.
- Restore notes automatically on revisit.
- Translate text snippets while reading.

## Scope note

This is a fun product experiment intended to explore practical browser-side utility rather than enterprise-scale distribution.
